CRUD Simples em PHP e MySQL

Este projeto é um exemplo básico de um sistema CRUD (Create, Read, Update, Delete) utilizando PHP e MySQL.
Foi desenvolvido com foco em simplicidade, para que iniciantes possam entender facilmente como funciona a interação entre o backend (PHP) e o banco de dados.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Funcionalidades: 

Criar novos registros

Listar registros existentes

Editar registros

Excluir registros

Conexão simples com banco de dados usando PDO

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Tecnologias Utilizadas:

PHP (PDO)

MySQL

HTML5

Servidor local como XAMPP, WAMP ou Laragon
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Como Usar:

Criar registro
Abra a página create.php e preencha o formulário.

Listar registros
Acesse o arquivo index.php, que mostra todos os registros cadastrados.

Editar registro
Clique em "Editar" na lista e altere os dados desejados.

Excluir registro
Clique em "Excluir" para remover o registro permanentemente.
